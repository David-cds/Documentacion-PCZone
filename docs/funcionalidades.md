# Funcionalidades del Proyecto
- Subida de imágenes
- Gestión de stock
- Clasificación por categorías

---

## Carrito de compra

El carrito utiliza cookies para almacenar productos.

### Características

- Añadir productos
- Eliminar productos
- Calcular total
- Controlar stock
- Reducir stock automáticamente

---

## Buscador de productos

El buscador permite encontrar productos similares al funcionamiento de PcComponentes.

### Ejemplos

- Buscar "gaming"
- Buscar "Ryzen"
- Buscar "portátil"
- Buscar "RTX"

---

## Filtros

El sistema permite filtrar productos por:

- Categoría
- Precio ascendente
- Precio descendente
- Orden alfabético

---

## Responsive Design

La página está desarrollada utilizando Mobile First.

### Adaptaciones

- Móviles
- Tablets
- Escritorio

---

## Slider automático

Se implementó un slider de imágenes usando JavaScript.

### Funciones

- Cambio automático de imágenes
- Diseño responsive
- Optimización visual

---

## AJAX

Se utiliza AJAX para cargar productos dinámicamente.

### Funcionalidades

- Carga sin recargar la página
- Ventana nueva con window.open()
- Mejora de experiencia de usuario
